  <h1 align="center">Qraken.jl</h1>
  <p align="center"><strong>Matrix-Free Quantum Simulator in Julia</strong></p>
</p>

---



**Qraken.jl** is a Julia package for simulating **pure states and mixed states** without constructing explicit operator matrices for GPU. It handles both closed and open quantum systems using bitwise operations on state amplitudes.
